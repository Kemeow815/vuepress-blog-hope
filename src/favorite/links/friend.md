::: tip 欢迎交换友情链接

在评论区按照如下格式留言:

```yaml
昵称: 网站或个人昵称
描述: 网站或个人简介
地址: 完整的主页地址
头像: 头像链接
```

然后将本站的信息添加到你的站点中

:::
