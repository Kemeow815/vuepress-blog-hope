## 一些其它指令

```bash

## npm 升级
npm -g install npm

## 清除缓存
npm cache clean --force

## 卸载

npm uninstall xxx (-g)


## 查看更新
npm -g outdated


## 其它指令

npm audit fix

npm audit fix --force

npm audit

```
